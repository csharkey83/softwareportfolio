Java Project


Project Description

For this project I had to create a console program in JAVA which would ask the user to select a temperature type they would like to convert from (from either Celsius, Fahrenheit, or Kelvin). Then the program would ask for the degrees (numeric value) that they would like the system to convert from. The program would then have to take the user inputs, calculate the conversion in the other temperature types not selected and display the results. It would continue to loop through this process until the user decided to quite the program by typing the letter "Q". In addition, the program would have to accept any letter case (upper or lower) and if a valid choice is not enter, it would just repeat the question until one of the 4 options was selected ("c", "f", "k" or "q").
